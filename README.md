## Description

[Fancy store API](https://github.com/sebastianupr/fancy-store-api) is a Node backend project, for Fancy Store App by @sebastianupr

## Technologies

- Language: [Node](https://nodejs.org/es/)

- Framework: [NestJS](https://nestjs.com/)

## Installation

```bash

$ npm install
# or
$ yarn dev

```

## Running the app

```bash

# development

$ npm run start


# watch mode

$ npm run start:dev



# production mode

$ npm run start:prod

```

## Test

```bash

# unit tests

$ npm run test



# e2e tests

$ npm run test:e2e



# test coverage

$ npm run test:cov

```
